📚 Tools & Features Guide

📤

Upload Data

Import participant information from CSV files. This is the first step to populate your participant database with names, emails, preferences, and other details.

Go to Upload Data →
🔍

Data & Filters

View all participant data in table format. Filter by gender, relationship style, main intention, venue preferences, and more. Search for specific participants and export data as needed.

Go to Data & Filters →
👥

Groups

View pre-organized groups based on relationship style and main intention. Groups are automatically categorized (6 women + 6 men for regular groups, 12 for LGBTQ+). Click on any person's name to view their full profile.

Go to Groups →
📅

Events

Manage events and create custom events. View event details organized by group type. Use the Custom Event box to search, add participants, and assign them to 4 primary slots or backup list. Each event shows venue voting and participant responsiveness ratings.

Go to Events →
📋

Polls

Create and manage polls for event scheduling and participant feedback. Set up poll details, add FAQ links, and track RSVP responses. Generate shareable poll links and monitor real-time vote counts with location preferences.

Go to Polls →
📊

Dashboard

Central hub for all event management. View statistics, track participant attendance and responses, see event participation data, and monitor 5-star responsiveness ratings. Manage event invitations, responses, and attendance records all in one place.

Go to Dashboard →

🚀 Getting Started

  • 1️⃣ Upload Data - Start by uploading your participant list with all their information
  • 2️⃣ Review Groups - Check the Groups section to see how participants are organized
  • 3️⃣ Create Events - Build custom events with your selected participants
  • 4️⃣ Create Polls - Send out polls to gather RSVPs and venue preferences
  • 5️⃣ Track in Dashboard - Monitor all responses and attendance in the Dashboard
